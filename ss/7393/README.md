### 문제
계단수 찾기 문제

### 풀이
* DP
* dp[자리수][숫자][bitmask]
* 숫자는 0으로 시작하지 않는다. 
* 시작 조건
* * 첫째 자리는 0으로 시작하지 않고 1~9로 시작하고 기본 계단수로 카운팅 해서 시작한다.
* 종료 조건
* * 자리수 도달
* 리턴
* * 비트플립이 다된 (1023) 것들만 카운팅 한다.

### 회고
