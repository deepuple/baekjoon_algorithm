1. 각 치킨집에서 집까지의 거리를 모두 구함, 이를테면 : chicken[1번 치킨집][1번집]=3;
2. 치킨집을 선택(조합으로)
3. 선택한 치킨집의 최소합을 추출 Math.min(chicken[1][1], chicken[2][1])+Math.min(chicken[1][2], chicken[2][2])
