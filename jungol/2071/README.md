# 문제 
파스칼 삼각형 출력   
m(크기), n(모양)이 주어질 때 다음과 같이 출력한다.   
```
n=1
1
11
121
1331
14641

n=2
1 4 6 4 1
 1 3 3 1
  1 2 1
   1 1
    1

n=3
1
41
631
4321
11111
```
# 풀이
파스칼 배열 만들어 놓고 출력만 바꾼다

```
arr[i][j] = 1 // 1,1에 1을 넣고...

for(i = 1; i <= m; i++)
  for(j = 1; j <= m; j++)
    arr[i][j] = arr[i-1][j-1]+arr[i-1][j]; //파스칼 배열 생성
```

출력 1  
```
for(i = 1; i <= m; i++)
  for(j = 1; j <= i; j++)
    cout << arr[i][j];
  cout << endl;
```
출력 2      
i 공백   
5  0   
4  1   
3  2   
2  3   
1  4 // n-i 수 만큼 공백이 들어감   
```
for(i = m; i >= 1; i--)
  for(j = 1; j >= m-i ; j++)
    cout << " ";
  for(j = 1; j <= i; j++)
    cout << arr[i][j];
  cout << endl;
```
출력 3   
(5,5)   
(5,4)(4,4)   
(5,3)(4,3)(3,3)   
(5,2)(4,2)(3,2)(2,2)   
(5,1)(4,1)(3,1)(2,1)(1,1)   
```
for(j = m ; j >= 1 ; j--)
  for(i = m ; i >= j ; i--)
    cout << arr[i][j];
cout << endl;
```
