# 문제
아래와 같이 출력할 것, n은 홀수이다.

n =  7 인 경우, 

```
*
 ***
  *****
   *******
  *****
 ***
*
```

# 풀이
```
*           // 공백 0, 출력 1
 ***        // 1, 3
  *****     // 2, 5
   *******  // 3, 7 --> i를 1~(n/2+1)줄 만큼 반복하면서, 공백은 i-1 만큼, 출력은 2i-1 만큼, 
   
  *****     //위에 줄 반전
 ***
*
```
```
for(i = 1; i<=(n/2)+1; i++)
  for(j = 1; j <= i-1 ; j++)
    cout << " ";
  for(j = 1; j <=(2*i)-1 ; j++)
    cout << "*";
  cout << endl;
  
for(i = (n/2); i>=1; i--)
  for(j = 1; j <= i-1 ; j++)
    cout << " ";
  for(j = 1; j <=(2*i)-1 ; j++)
    cout << "*";
  cout << endl;
```
